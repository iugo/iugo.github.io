远程办公申请

申请

因工作性质与个人原因, 特此提交远程办公申请. 请领导予以考虑.

  • 远程办公即在传统办公环境以外利用现代化沟通方式进行办公.

工作原因(性质)

成熟的远程办公管理工具

团队协作, 版本控制, 即时通讯

在互联网大环境下, 催生了许多团队协作的方案, 如果执行力高, 则能更好地提升效率, 总结经验, 规划发展.

目前公司所有的 Teambition 项目协作工具即为我推荐的基于互联网的协作平台. 可以有效规划任务, 传递信息.

团队协作工具减少了沟通成本, 让远程办公成为可能.

写代码是一个持续性的过程, 在形成完整功能前, 需要不断添加细小的功能, 修正错误. 为了方便管理, 我主导在技术工作中使用 Git 作为主要的版本控制工具.

目前技术部门使用基于 Git 的 coding.net 服务进行代码托管与演示部署.

Git 这样的远程代码仓库让代码工作可以异地、离线完成, 是目前主流的程序代码管理方式. 跨过技术团队的成员在不同时区的不同国家, 都使用该工具进行协作, 良好地完成任务.

依靠良好的宽带与五花八门的沟通工具, 目前的互联网环境已经允许身处异地的人们「面对面」交流, 距离再也不是交流的障碍.
